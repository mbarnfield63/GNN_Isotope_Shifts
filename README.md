# Graph Neural Network for Isotopic Shift Corrections

**Aim**: Improve upon MLP method documented in recent [paper](https://doi.org/10.1016/j.jms.2026.112084).
**Plan**: Utilise GNN architecture to create full graph of all isotopologues within a molecule (and possibly across molecules).
**Initial**: Implement over diatomics, specifically CO for comparisons to original paper.
